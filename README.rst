To install, use

.. code-block:: console

    $ pip install --user -e .

Once installed, use

.. code-block:: console

    $ algosim -vv [-c CONFIG]

to run a simulation with full debugging info.

Using the seed in the current config file, run

.. code-block:: console

    $ algosim --plot "recognize challenge"

to see graph of a given, high quality video's rating over time.
